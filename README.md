# nameless
You can calculate (p+q)^2, (p-q)^2
because (p+q)^2 = p^2+q^2+2pq = p^2+q^2+2*n
    and (p-q)^2 = p^2+q^2-2pq = p^2+q^2-2*n
    
Once you know them, two values above, you can easily calculate p, q
because p+q=p_p_q
    and p-q=p_m_q
